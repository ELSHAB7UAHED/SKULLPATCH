# الأسئلة الشائعة - mo7aoil

هذا الملف يجيب على الأسئلة الأكثر شيوعاً حول مشروع mo7aoil.

## 🤔 أسئلة عامة

### ما هو mo7aoil؟
mo7aoil هو موقع احترافي وعصري لمعالجة الصور والملفات، يوفر أدوات متقدمة لضغط الصور، تحويل صيغ الملفات، معالجة PDF، والفيديو، وأدوات متقدمة أخرى.

### لماذا اسم "mo7aoil"؟
"mo7aoil" يعني "المحول" أو "المحول" باللغة العربية، ويعكس هدف المشروع في تحويل ومعالجة الملفات بطرق مختلفة.

### هل المشروع مجاني؟
نعم، المشروع مجاني ومفتوح المصدر. يمكنك استخدامه، تعديله، ونشره بحرية وفقاً لترخيص MIT.

### ما هي اللغات المدعومة؟
حالياً يدعم المشروع اللغة العربية بشكل أساسي، مع خطط لدعم لغات أخرى في المستقبل.

## 🚀 التثبيت والتشغيل

### ما هي المتطلبات الأساسية؟
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn
- متصفح حديث يدعم ES6+

### كيف أقوم بتشغيل المشروع؟
```bash
# الطريقة السريعة
./start.sh

# الطريقة التقليدية
npm install
npm run build
npm start
```

### ما هو المنفذ الافتراضي؟
المنفذ الافتراضي هو 3000. يمكنك الوصول للموقع على `http://localhost:3000`

### كيف أغير المنفذ؟
```bash
# في ملف server.js
const PORT = process.env.PORT || 3001;

# أو في Terminal
PORT=3001 npm start
```

### لماذا لا يعمل المشروع؟
تحقق من:
1. تثبيت Node.js بشكل صحيح
2. تثبيت التبعيات: `npm install`
3. عدم استخدام المنفذ 3000 من تطبيق آخر
4. مراجعة رسائل الخطأ في Terminal

## 🛠️ الميزات والأدوات

### ما هي الأدوات المتاحة؟
- **أدوات الصور**: ضغط، تغيير الحجم، تحويل الصيغ، علامة مائية
- **أدوات PDF**: دمج، تقسيم، تحويل إلى صورة
- **أدوات الفيديو**: ضغط، تحويل
- **أدوات متقدمة**: معالجة مجمعة، OCR

### هل يمكن معالجة ملفات كبيرة؟
نعم، يدعم المشروع ملفات حتى 100MB. يمكن تعديل هذا الحد في إعدادات الخادم.

### ما هي صيغ الملفات المدعومة؟
- **الصور**: JPG, PNG, GIF, WebP, SVG, TIFF
- **PDF**: PDF
- **الفيديو**: MP4, AVI, MOV, WMV, FLV
- **المستندات**: DOC, DOCX, TXT, RTF

### هل يمكن معالجة ملفات متعددة؟
نعم، يمكن رفع ومعالجة عدة ملفات في نفس الوقت.

### كيف أستخدم Docker؟
```bash

# تشغيل الحاوية
```

## 🚨 استكشاف الأخطاء

### المشكلة: الملف لا يرفع
**الحلول**:
1. تأكد من أن الملف أصغر من 100MB
2. تأكد من أن صيغة الملف مدعومة
3. أعد تحميل الصفحة
4. تحقق من اتصال الإنترنت

### المشكلة: المعالجة بطيئة
**الحلول**:
1. انتظر قليلاً للملفات الكبيرة
2. تأكد من أن الخادم يعمل بشكل صحيح
3. تحقق من استخدام CPU و RAM

### المشكلة: خطأ في البناء
**الحلول**:
```bash
# امسح الكاش
npm run clean

# أعد تثبيت التبعيات
rm -rf node_modules package-lock.json
npm install

# أعد البناء
npm run build
```

### المشكلة: المنفذ مشغول
**الحلول**:
```bash
# ابحث عن العمليات
lsof -i :3000

# أوقف العملية
kill -9 <PID>

# أو غير المنفذ
PORT=3001 npm start
```
