# 🚀 mo7aoil - موقع احترافي وعصري لمعالجة الصور والملفات

موقع `mo7aoil` هو منصة احترافية وعصرية جداً لمعالجة الصور والملفات، مصمم بأحدث التقنيات وأفضل الممارسات في تطوير الويب.

## ✨ المميزات الرئيسية

### 🎨 تصميم عصري واحترافي
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة مستخدم حديثة وسهلة الاستخدام
- ألوان متدرجة وتأثيرات بصرية مذهلة
- خطوط عربية احترافية (Cairo)
- رسوم متحركة سلسة وجذابة

### 🛠️ أدوات قوية ومتطورة
- **معالجة الصور**: ضغط، تغيير الحجم، تحويل الصيغ، إضافة علامات مائية
- **أدوات PDF**: دمج، تقسيم، تحويل إلى صور
- **معالجة الفيديو**: ضغط وتحويل الصيغ
- **أدوات متقدمة**: معالجة مجمعة، استخراج النصوص (OCR)

### 🚀 تقنيات حديثة
- Node.js مع Express.js
- Socket.IO للاتصال المباشر
- معالجة الملفات باستخدام Sharp و Jimp
- معالجة PDF باستخدام pdf-lib
- أمان عالي مع Helmet و CORS
- ضغط البيانات مع Compression

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd mo7aoil
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **تشغيل المشروع**
```bash
# وضع التطوير
npm run dev

# وضع الإنتاج
npm start
```

4. **فتح المتصفح**
```
http://localhost:3000
```

## 🛠️ الأوامر المتاحة

```bash
npm start          # تشغيل الخادم
```

## 🌐 API Endpoints

### معالجة الصور
- `POST /api/compress-image` - ضغط الصور
- `POST /api/resize-image` - تغيير حجم الصور
- `POST /api/convert-image` - تحويل صيغ الصور
- `POST /api/add-watermark` - إضافة علامة مائية

### معالجة PDF
- `POST /api/merge-pdfs` - دمج ملفات PDF
- `POST /api/split-pdf` - تقسيم ملف PDF
- `POST /api/pdf-to-image` - تحويل PDF إلى صور

### معالجة الفيديو
- `POST /api/compress-video` - ضغط الفيديو
- `POST /api/convert-video` - تحويل الفيديو

### أدوات متقدمة
- `POST /api/batch-process` - معالجة مجمعة
- `POST /api/cleanup` - تنظيف الملفات المؤقتة
