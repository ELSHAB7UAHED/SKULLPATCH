## 🚨 المشاكل الشائعة

### مشاكل التثبيت
```bash
# خطأ: webpack: command not found
npm install -g webpack-cli

# خطأ: EACCES: permission denied
sudo npm install

# خطأ: Python not found
npm install --python=python3
```

### مشاكل التشغيل
```bash
# خطأ: Port already in use
# تغيير المنفذ في server.js
const PORT = process.env.PORT || 3001;

# خطأ: Cannot find module
rm -rf node_modules package-lock.json
npm install

# خطأ: Memory limit
node --max-old-space-size=4096 server.js
```

### مشاكل البناء
```bash
# خطأ: Build failed
npm run clean
npm install
npm run build

# خطأ: CSS not loading
# تأكد من أن webpack.config.js صحيح
# تأكد من أن الملفات في المجلد الصحيح
```

## 🛠️ استكشاف الأخطاء

### خطوات عامة
1. **أعد تشغيل الخادم**: `npm start`
2. **امسح الكاش**: `npm run clean`
3. **أعد تثبيت التبعيات**: `npm install`
4. **أعد بناء المشروع**: `npm run build`
5. **تحقق من السجلات**: راجع console للرسائل

### أدوات التشخيص
```bash
# فحص التبعيات
npm audit

# فحص الإصدارات
npm outdated

# فحص الحجم
npm list --depth=0

# فحص الأداء
npm run build --profile
```
