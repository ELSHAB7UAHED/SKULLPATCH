## 🚀 بدء العمل

### 1. استنساخ المشروع
```bash
git clone https://github.com/username/mo7aoil.git
cd mo7aoil
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إنشاء branch جديد
```bash
git checkout -b feature/your-feature-name
# أو
git checkout -b fix/your-bug-fix
```

### 4. تطوير الميزة
```bash
# تشغيل في وضع التطوير
npm run dev

# بناء المشروع
npm run build

# تشغيل الاختبارات
npm test
```

### 5. رفع التغييرات
```bash
git add .
git commit -m "feat: إضافة ميزة جديدة"
git push origin feature/your-feature-name
```

### 6. إنشاء Pull Request
- اذهب إلى [GitHub](https://github.com/username/mo7aoil)
- انقر على "New Pull Request"
- اختر branch الميزة
- اكتب وصفاً مفصلاً للتغييرات
